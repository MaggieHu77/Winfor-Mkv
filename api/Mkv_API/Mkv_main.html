set_params1()[源代码]¶
回测模式的权重优化及回测结果计算、写出.
根据不同设置初始组合控件的方法提取有效参数.
创建Mkv_pointwise_codes.BTcodes类对象，用于生成回测期间需要进入组合优化的有效股票.
遍历各个调仓日期，计算待配权股票在统计期内的均值和协方差矩阵.
推送统计量和参数进入mosek进行Markowitz组合寻优.
模拟回测期间的持仓并记录和写出回测结果.

Mkv_main.dict2list(stock_pannel, var_pannel, suffix=None, date_first=False)[源代码]¶

这个函数扩展性不佳，为了分隔较复杂的数据形式重组功能.

参数:
  • stock_pannel – 包含回测全部权股票基本信息的字典数据.
  • var_pannel – 包含回测全部权重信息的字典数据.
  • suffix – 需要附加在var_pannel元素后的list.
返回:

通过pd.DataFrame merge各个不同日期回测结果后转换为可以直接写入workbook的list形式.
Mkv_main.lookup_w(w_panel, t, pre_w=None, pre_r=None)[源代码]¶

用于返回对应高频（这里是daily）日期在回测期间的权重（调频周期为低频周、月）.

参数:
  • w_panel – dict，在调仓日（收盘后）的对应新权重.
  • t – str, 需要确定组合日收益率的当日.
  • pre_w – list, 前2交易日收盘后的权重，即是前1交易日开盘前权重.
  • pre_r – list, 前1交易日当天收益率.
返回:

list,t当日开盘前权重.
