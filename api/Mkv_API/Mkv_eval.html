eval_returns(var='pct_chg', get_t_seq=False)[源代码]¶

在各时间段内持仓股票品种不变的情形下，应用该函数获取每个时点上每只股票对应频率的变量值.

参数:
  • var – str,wind变量名，一般是区间收益率.
  • get_t_seq – bool,是否返回时间序列字符串.
返回:

变量集合，日期list.
eval_returns12(codes, eval_date, var='pct_chg', eval_interval_start=None, include_head=False)[源代码]¶

返回特定评估日，所有股票的区间收益率.

参数:
  • codes – list, stock codes.
  • eval_date – str, 评估日当日， fmt=’%Y-%m-%d’或’%Y-%m-%d H:M:S’.
  • var – str或者list, 需要提取的变量，一般是收益率.
  • eval_interval_start – str，区间开始日期，fmt=’%Y-%m-%d’，只有_freq=H时有效,
  • skip – 一次评估基础K线长度，特别用于_freq=H.
  • include_head – 数据开始是否包括左侧区间点.
返回:

list or DataFrame，股票变量对象.
